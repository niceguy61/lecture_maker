# Day 10: RDS (Relational Database Service)

## 📚 학습 개요

AWS의 관리형 관계형 데이터베이스 서비스인 RDS를 학습합니다.
다양한 데이터베이스 엔진의 특징을 이해하고, 고가용성과 성능을 보장하는 데이터베이스 아키텍처를 구축하는 방법을 익힙니다.

## 🎯 학습 목표

- RDS 엔진별 특징과 선택 기준 이해
- Multi-AZ 및 Read Replica 구성 방법 습득
- 데이터베이스 백업 및 복원 전략 학습
- RDS 성능 모니터링 및 튜닝 능력 개발
