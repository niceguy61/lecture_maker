# Day 12: 데이터 마이그레이션 서비스

## 📚 학습 개요

AWS의 다양한 데이터 마이그레이션 서비스를 학습합니다.
온프레미스에서 클라우드로, 또는 클라우드 간 데이터 이동을 위한 최적의 전략과 도구를 선택하는 방법을 익힙니다.

## 🎯 학습 목표

- AWS DMS를 활용한 데이터베이스 마이그레이션 방법 습득
- DataSync를 통한 파일 시스템 마이그레이션 이해
- Storage Gateway의 하이브리드 스토리지 활용 방안 학습
- Snow 패밀리를 통한 대용량 데이터 전송 전략 개발
