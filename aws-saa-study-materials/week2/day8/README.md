# Day 8: Amazon S3 (Simple Storage Service)

## 📚 학습 개요
Amazon S3의 핵심 개념부터 고급 기능까지 체계적으로 학습하는 날입니다. 객체 스토리지의 특성, 다양한 스토리지 클래스, 보안 설정, 그리고 성능 최적화 방법을 실습과 함께 익혀보겠습니다.

## 🎯 학습 목표
- [ ] Amazon S3의 핵심 개념과 아키텍처 이해
- [ ] S3 스토리지 클래스별 특성과 사용 사례 파악
- [ ] S3 보안 및 액세스 제어 방법 학습
- [ ] 버전 관리 및 라이프사이클 정책 활용
- [ ] S3 성능 최적화 및 비용 관리 방법 습득
- [ ] 정적 웹사이트 호스팅 실습 완료
