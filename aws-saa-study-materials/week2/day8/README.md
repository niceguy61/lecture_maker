# Day 8: S3 (Simple Storage Service)

## 📚 학습 개요

AWS의 핵심 스토리지 서비스인 S3를 학습합니다.
객체 스토리지의 개념부터 고급 기능까지, 확장 가능하고 내구성 있는 스토리지 솔루션을 구축하는 방법을 익힙니다.

## 🎯 학습 목표

- S3 버킷 및 객체 관리 방법 습득
- 스토리지 클래스별 특징과 사용 사례 이해
- 생명주기 정책을 통한 비용 최적화 방안 학습
- S3 보안 및 액세스 제어 구현 능력 개발
