# Day 14: 주간 복습 및 실습 프로젝트

## 📚 학습 개요

Week 2의 스토리지 및 데이터베이스 서비스 내용을 종합적으로 복습하고, 
실제 데이터 레이크 아키텍처를 구축하는 실습 프로젝트를 통해 학습 내용을 통합 적용합니다.

## 🎯 학습 목표

- Week 2 핵심 개념 종합 정리 및 복습
- 데이터 레이크 아키텍처 설계 및 구현 능력 개발
- 스토리지 서비스 간 연동 및 최적화 방법 습득
- 실무 적용 가능한 데이터 관리 전략 수립

## 📖 학습 내용

### 1. 주간 복습
- [Week 2 핵심 개념 정리](./theory.md)
- S3, EBS, EFS, RDS, DynamoDB 비교 분석
- 데이터 마이그레이션 및 백업 전략 요약
- 스토리지 비용 최적화 방안

### 2. 실습 프로젝트
- [데이터 레이크 아키텍처 구축](./hands-on/)
- S3 기반 데이터 저장소 설계
- Lambda를 활용한 데이터 처리 파이프라인
- Athena를 통한 데이터 분석 환경 구성

### 3. 종합 평가
- [주간 퀴즈 (35문제)](./quiz.md)
- 스토리지 서비스 비교 치트 시트 작성
- 비용 최적화 시나리오 분석

- **스토리지 아키텍처 설계**: 요구사항에 맞는 최적의 스토리지 솔루션 선택 능력
