# Day 16: CloudFront 및 CDN

## 📚 학습 개요

AWS의 글로벌 콘텐츠 전송 네트워크인 CloudFront를 학습합니다.
전 세계 사용자에게 빠르고 안전한 콘텐츠 배포를 위한 CDN 구성과 최적화 방법을 익힙니다.
