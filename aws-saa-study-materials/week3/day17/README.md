## 📊 핵심 개념 요약

### DNS 레코드 타입
- **A**: IPv4 주소 매핑
- **AAAA**: IPv6 주소 매핑
- **CNAME**: 도메인 별칭
- **MX**: 메일 서버 정보
- **TXT**: 텍스트 정보

### Route 53 라우팅 정책
1. **Simple**: 기본 라우팅
2. **Weighted**: 가중치 기반 분산
3. **Latency-based**: 지연 시간 최적화
4. **Failover**: 장애 조치
5. **Geolocation**: 지리적 위치 기반
6. **Geoproximity**: 지리적 근접성
7. **Multivalue**: 다중값 응답
