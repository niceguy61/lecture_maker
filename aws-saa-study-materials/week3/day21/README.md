## 🎯 주요 학습 성과

- **확장 가능한 아키텍처**: 로드 밸런서와 오토 스케일링을 활용한 탄력적 시스템 구축
- **글로벌 서비스**: CloudFront와 Route 53을 통한 전 세계 사용자 대상 서비스 제공
- **서버리스 개발**: Lambda와 API Gateway를 활용한 이벤트 기반 애플리케이션 개발
- **컨테이너 오케스트레이션**: ECS/EKS를 통한 컨테이너 기반 애플리케이션 관리
- **자동화된 배포**: CI/CD 파이프라인을 통한 안전하고 효율적인 배포 프로세스 구축
