# AWS Solutions Architect Associate (SAA-C03) 학습 자료

## 📚 과정 개요

AWS Solutions Architect Associate (SAA-C03) 시험 합격을 위한 체계적인 4주 학습 프로그램입니다. 
대학교 막학기 컴퓨터공학과 학생 수준에 맞춰 이론 학습, 실습, 그리고 진도 확인을 통합한 학습 플랫폼을 제공합니다.

## 🎯 학습 목표

- AWS 핵심 서비스에 대한 깊이 있는 이해
- 실무에 적용 가능한 아키텍처 설계 능력 습득
- SAA-C03 시험 합격을 위한 체계적인 준비
- Python을 활용한 AWS 서비스 실습 경험

## � 시작하기

### 1. 학습 환경 준비
- Python 3.8+ 설치
- AWS CLI 설치 및 구성
- 텍스트 에디터 또는 IDE 준비

### 2. 학습 방법
1. **이론 학습**: 각 day의 `theory.md` 파일로 개념 학습
2. **시각화 자료**: `visuals/` 폴더의 다이어그램으로 이해 보강
3. **실습**: `hands-on/` 폴더의 Python 스크립트로 실무 경험
4. **퀴즈**: `quiz.md`로 학습 내용 점검
5. **진도 추적**: `resources/progress-tracker.md`로 학습 현황 관리

### 3. 권장 학습 시간
- **평일**: 2-3시간 (이론 1시간, 실습 1-2시간)
- **주말**: 3-4시간 (복습 및 프로젝트)
- **총 학습 시간**: 주당 15-20시간
