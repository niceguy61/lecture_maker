# AWS 서비스 개요 치트 시트

## 🖥️ 컴퓨팅 서비스

### EC2 (Elastic Compute Cloud)
- **용도**: 가상 서버 인스턴스
- **특징**: 다양한 인스턴스 타입, 온디맨드/예약/스팟 인스턴스
- **사용 사례**: 웹 서버, 애플리케이션 서버, 개발 환경

### Lambda
- **용도**: 서버리스 컴퓨팅
- **특징**: 이벤트 기반 실행, 자동 스케일링, 사용한 만큼 과금
- **사용 사례**: API 백엔드, 데이터 처리, 자동화 작업

### ECS (Elastic Container Service)
- **용도**: 컨테이너 오케스트레이션
- **특징**: Docker 컨테이너 관리, Fargate 지원
- **사용 사례**: 마이크로서비스, 컨테이너화된 애플리케이션

### EKS (Elastic Kubernetes Service)
- **용도**: 관리형 Kubernetes
- **특징**: Kubernetes 클러스터 관리, AWS 서비스 통합
- **사용 사례**: 복잡한 컨테이너 워크로드, 멀티 클라우드

## 💾 스토리지 서비스

### S3 (Simple Storage Service)
- **용도**: 객체 스토리지
- **특징**: 무제한 용량, 11 9's 내구성, 다양한 스토리지 클래스
- **사용 사례**: 웹사이트 호스팅, 백업, 데이터 아카이브

### EBS (Elastic Block Store)
- **용도**: 블록 스토리지
- **특징**: EC2 인스턴스 연결, 다양한 볼륨 타입
- **사용 사례**: 데이터베이스, 파일 시스템, 부트 볼륨

### EFS (Elastic File System)
- **용도**: 네트워크 파일 시스템
- **특징**: NFSv4, 자동 확장, 다중 AZ 접근
- **사용 사례**: 공유 스토리지, 콘텐츠 관리, 웹 서빙

### Glacier
- **용도**: 아카이브 스토리지
- **특징**: 저비용, 장기 보관, 다양한 검색 옵션
- **사용 사례**: 백업, 아카이브, 규정 준수

## 🗄️ 데이터베이스 서비스

### RDS (Relational Database Service)
- **용도**: 관리형 관계형 데이터베이스
- **지원 엔진**: MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, Aurora
- **특징**: 자동 백업, Multi-AZ, Read Replica
- **사용 사례**: 웹 애플리케이션, ERP, CRM

### DynamoDB
- **용도**: NoSQL 데이터베이스
- **특징**: 완전 관리형, 자동 스케일링, 밀리초 지연 시간
- **사용 사례**: 모바일 앱, 게임, IoT, 실시간 애플리케이션

### ElastiCache
- **용도**: 인메모리 캐싱
- **지원 엔진**: Redis, Memcached
- **특징**: 마이크로초 지연 시간, 고가용성
- **사용 사례**: 세션 스토어, 캐싱, 실시간 분석

### Redshift
- **용도**: 데이터 웨어하우스
- **특징**: 컬럼형 스토리지, 병렬 처리, SQL 지원
- **사용 사례**: 비즈니스 인텔리전스, 데이터 분석

## 🌐 네트워킹 서비스

### VPC (Virtual Private Cloud)
- **용도**: 가상 네트워크
- **특징**: 논리적 격리, 사용자 정의 네트워크 구성
- **구성 요소**: 서브넷, 라우팅 테이블, 인터넷 게이트웨이, NAT

### CloudFront
- **용도**: CDN (Content Delivery Network)
- **특징**: 글로벌 엣지 로케이션, 캐싱, DDoS 보호
- **사용 사례**: 웹사이트 가속, 동영상 스트리밍, API 가속

### Route 53
- **용도**: DNS 서비스
- **특징**: 도메인 등록, DNS 라우팅, 헬스 체크
- **라우팅 정책**: Simple, Weighted, Latency, Failover, Geolocation

### API Gateway
- **용도**: API 관리 서비스
- **특징**: RESTful API, WebSocket, 인증/권한 부여
- **사용 사례**: 마이크로서비스 API, 서버리스 백엔드

## 🔒 보안 서비스

### IAM (Identity and Access Management)
- **용도**: 권한 및 액세스 관리
- **구성 요소**: 사용자, 그룹, 역할, 정책
- **특징**: 세밀한 권한 제어, MFA 지원

### KMS (Key Management Service)
- **용도**: 암호화 키 관리
- **특징**: 하드웨어 보안 모듈, 키 순환, 감사 로그
- **사용 사례**: 데이터 암호화, 규정 준수

### CloudTrail
- **용도**: API 호출 로깅
- **특징**: 모든 AWS API 호출 기록, 감사 추적
- **사용 사례**: 보안 분석, 규정 준수, 문제 해결

### GuardDuty
- **용도**: 위협 탐지
- **특징**: 머신러닝 기반, 자동 위협 탐지
- **사용 사례**: 보안 모니터링, 이상 행위 탐지

## 📊 모니터링 및 관리

### CloudWatch
- **용도**: 모니터링 및 관찰성
- **기능**: 메트릭, 로그, 알람, 대시보드
- **사용 사례**: 성능 모니터링, 자동 스케일링, 문제 해결

### CloudFormation
- **용도**: 인프라스트럭처 as 코드
- **특징**: JSON/YAML 템플릿, 스택 관리
- **사용 사례**: 인프라 자동화, 환경 복제, 버전 관리

### Systems Manager
- **용도**: 운영 관리
- **기능**: 패치 관리, 구성 관리, 파라미터 스토어
- **사용 사례**: 서버 관리, 자동화, 규정 준수

## 💰 비용 관리

### Cost Explorer
- **용도**: 비용 분석 및 시각화
- **특징**: 사용량 패턴 분석, 비용 예측
- **사용 사례**: 비용 최적화, 예산 계획

### Budgets
- **용도**: 예산 설정 및 알림
- **특징**: 사용자 정의 예산, 임계값 알림
- **사용 사례**: 비용 제어, 예산 관리

### Trusted Advisor
- **용도**: 모범 사례 권장 사항
- **영역**: 비용 최적화, 성능, 보안, 내결함성
- **사용 사례**: 최적화 기회 식별, 보안 개선

## 📝 시험 팁

### 자주 출제되는 서비스 조합
- **웹 애플리케이션**: EC2 + ELB + RDS + S3
- **서버리스**: Lambda + API Gateway + DynamoDB
- **데이터 분석**: S3 + Redshift + QuickSight
- **하이브리드**: VPC + VPN/Direct Connect + Storage Gateway

### 서비스 선택 기준
- **비용**: 온디맨드 vs 예약 vs 스팟
- **성능**: 지연 시간, 처리량, IOPS
- **가용성**: Single-AZ vs Multi-AZ
- **확장성**: 수직 vs 수평 확장
