# Week 1 Day 7: 주간 복습 및 종합 실습

## 📅 학습 개요

Week 1의 마지막 날로, 지난 6일간 학습한 AWS 기초 서비스들을 종합적으로 복습하고 실제 3-tier 아키텍처를 구축하는 실습을 진행합니다.

## 🎯 학습 목표

- Week 1에서 학습한 모든 AWS 서비스의 통합적 이해
- 실제 운영 환경 수준의 3-tier 아키텍처 구축 경험
- 서비스 간 연관성과 의존성 파악
- 보안 모범 사례 적용 능력 향상

## 📚 학습 내용

### 1. 이론 복습 (45분)
- **파일**: `theory.md`
- **내용**: Week 1 전체 내용 요약 및 서비스 간 연관성
- **핵심 포인트**:

  - EC2 및 스토리지 서비스
  - VPC 네트워킹 개념
  - 통합 아키텍처 패턴

### 2. 종합 실습 (2-3시간)
- **파일**: `hands-on/setup-guide.md`
- **프로젝트**: 3-tier 웹 애플리케이션 아키텍처 구축
- **구성 요소**:
  - VPC 및 다중 AZ 서브넷 설계
  - 보안 그룹 및 NACL 구성
  - EC2 인스턴스 배포 (웹/앱/DB 티어)

### 3. 종합 퀴즈 (30분)

- **난이도**: 중급 (SAA-C03 수준)
- **합격 기준**: 80% (12문제 이상)
