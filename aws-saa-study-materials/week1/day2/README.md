# Day 2: IAM (Identity and Access Management)

## 📚 학습 개요

AWS의 보안 및 접근 제어 서비스인 IAM을 학습합니다. 
사용자, 그룹, 역할, 정책을 통한 세밀한 권한 관리와 보안 모범 사례를 익힙니다.

- IAM의 핵심 구성 요소 이해

- MFA 및 보안 강화 방안 학습
- IAM 정책 작성 및 관리 능력 개발
