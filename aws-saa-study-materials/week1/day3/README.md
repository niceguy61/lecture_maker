## 🎯 핵심 개념 요약

### EC2 기본 개념
- **EC2**: 클라우드의 가상 서버 서비스
- **인스턴스**: 실행 중인 가상 서버
- **AMI**: 인스턴스 시작용 템플릿
- **인스턴스 타입**: 하드웨어 사양 정의

### 보안 및 접근 관리
- **보안 그룹**: 가상 방화벽 (상태 저장)
- **키 페어**: SSH/RDP 접속용 암호화 키
- **IAM 역할**: 인스턴스에 AWS 권한 부여

### 네트워킹
- **퍼블릭 IP**: 인터넷 접근 가능한 IP
- **프라이빗 IP**: VPC 내부 통신용 IP
- **Elastic IP**: 고정 퍼블릭 IP

### 스토리지
- **EBS**: 네트워크 연결 영구 스토리지
- **Instance Store**: 물리적 연결 임시 스토리지
