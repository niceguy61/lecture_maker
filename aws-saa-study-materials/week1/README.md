# Week 1: AWS 기초 및 핵심 서비스

## 📚 주차 개요

AWS의 기본 개념과 핵심 서비스들을 학습하는 첫 번째 주입니다. 
AWS 글로벌 인프라부터 시작해서 IAM, EC2, VPC까지 AWS 아키텍처의 기초를 다집니다.

## 🎯 주차 학습 목표

- AWS 글로벌 인프라와 기본 개념 이해
- IAM을 통한 보안 및 권한 관리 숙달
- EC2 인스턴스 생성, 관리, 최적화 방법 습득
- VPC를 활용한 네트워크 아키텍처 설계 능력 개발

이 주를 마치면 다음을 할 수 있게 됩니다:
