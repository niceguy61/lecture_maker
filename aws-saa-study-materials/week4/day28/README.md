# Day 28: 시험 준비 및 팁

## 📚 학습 개요

SAA-C03 시험 응시를 위한 최종 준비 단계입니다.
시험 당일 준비사항부터 응시 전략까지, 성공적인 자격증 취득을 위한 마지막 점검을 진행합니다.

## 🎯 학습 목표

- 시험 당일 준비사항 및 체크리스트 완성
- 마지막 핵심 개념 복습 및 정리
- 시험 응시 전략 및 시간 관리 방법 숙지
- 자격증 취득 후 계획 수립

### 3. 시험 전략

## 🎯 시험 응시 전략

### 1. 시간 관리
- **총 시험 시간**: 130분 (2시간 10분)
- **문제 수**: 65문제
- **문제당 평균 시간**: 2분
- **검토 시간**: 마지막 10-15분 확보

### 2. 문제 해결 접근법
1. **문제 읽기**: 핵심 키워드와 요구사항 파악
2. **선택지 분석**: 명백히 틀린 답부터 제거
3. **AWS 서비스 매칭**: 요구사항에 가장 적합한 서비스 선택
4. **비용 효율성**: 동일한 기능이면 더 저렴한 옵션 선택

### 3. 주의사항
- **실무 경험 우선**: 이론보다는 실제 구현 경험 기반 선택
- **최신 서비스**: 새로운 서비스보다는 검증된 서비스 우선
- **과도한 엔지니어링 지양**: 단순하고 직접적인 솔루션 선택

## 🏆 성공을 위한 마지막 조언

1. **충분한 휴식**: 시험 전날 충분한 수면
2. **자신감**: 4주간의 체계적인 학습에 대한 믿음
3. **침착함**: 모르는 문제가 나와도 당황하지 말고 차근차근 접근
4. **시간 관리**: 어려운 문제는 표시 후 나중에 해결
