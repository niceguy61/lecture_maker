# Day 22: CloudWatch 및 모니터링

## 📚 학습 개요

AWS의 포괄적인 모니터링 서비스인 CloudWatch를 학습합니다.
메트릭, 로그, 이벤트를 통한 시스템 모니터링과 X-Ray를 활용한 분산 추적으로 운영 가시성을 확보하는 방법을 익힙니다.

## 🎯 학습 목표

- CloudWatch 메트릭, 로그, 이벤트 활용 방법 습득
- 커스텀 메트릭 및 대시보드 구성 능력 개발
- CloudWatch Alarms와 SNS 연동을 통한 알림 시스템 구축
- X-Ray를 활용한 분산 애플리케이션 추적 방법 이해
