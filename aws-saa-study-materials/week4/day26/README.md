# Day 26: 종합 실습 프로젝트

## 📚 학습 개요

4주간 학습한 모든 AWS 서비스를 통합하여 엔터프라이즈급 3-tier 애플리케이션을 구축합니다.
보안, 모니터링, 백업을 포함한 완전한 아키텍처를 설계하고 구현하는 종합 실습을 진행합니다.

## 🎯 학습 목표

- 전체 과정 통합 프로젝트를 통한 실무 역량 개발
- 엔터프라이즈급 3-tier 애플리케이션 아키텍처 구축
- 보안, 모니터링, 백업이 포함된 완전한 솔루션 설계
- 비용 최적화 및 성능 튜닝 실무 경험 습득

## 📖 학습 내용

### 1. 프로젝트 계획
- [종합 아키텍처 설계](./theory.md)
- 요구사항 분석 및 서비스 선택
- 보안 및 규정 준수 고려사항
- 비용 및 성능 최적화 전략

### 2. 아키텍처 구성도
- [엔터프라이즈 아키텍처 다이어그램](./visuals/)
- 네트워크 토폴로지 설계
- 보안 계층 구조도

### 3. 종합 실습
- [완전한 3-tier 애플리케이션 구축](./hands-on/)
- VPC, 서브넷, 보안 그룹 설정
- ALB, Auto Scaling, RDS Multi-AZ 구성
- CloudFront, Route 53 글로벌 배포
- CloudWatch, CloudTrail 모니터링 설정

### 4. 프로젝트 검증
- [아키텍처 검토 체크리스트](./quiz.md)

## ⏰ 예상 학습 시간

**총 4-5시간**
- 설계 및 계획: 1시간
- 실습 구현: 3시간
- 테스트 및 검증: 1시간

## 📋 체크리스트

- [ ] 전체 아키텍처 설계 완료
- [ ] VPC 및 네트워크 구성 완료
- [ ] 웹 계층 (ALB, Auto Scaling) 구축 완료
- [ ] 애플리케이션 계층 (EC2, ECS) 구축 완료
- [ ] 데이터베이스 계층 (RDS, DynamoDB) 구축 완료
- [ ] 보안 설정 (IAM, Security Groups) 완료
- [ ] 모니터링 시스템 (CloudWatch) 구축 완료
- [ ] 백업 및 재해 복구 설정 완료

## 🎯 프로젝트 성과

- **완전한 엔터프라이즈 아키텍처**: 실무에서 바로 적용 가능한 확장 가능한 시스템
- **보안 강화**: 다층 보안 아키텍처 및 모니터링 시스템
- **운영 자동화**: 모니터링, 백업, 스케일링 자동화
- **비용 최적화**: Reserved Instances, Spot Instances 활용
